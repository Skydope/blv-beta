# Bolívar Digital 2030 - Ciudad Inteligente

Una aplicación web moderna y tecnológica que presenta el proyecto de transformación digital de San Carlos de Bolívar en una ciudad inteligente.

## Características

- **SPA (Single Page Application)** construida con Astro, Tailwind CSS y TypeScript
- **Diseño moderno y tecnológico** con animaciones fluidas y efectos visuales
- **Totalmente responsive** optimizado para dispositivos móviles y desktop
- **Modal interactivo** con el contenido completo del proyecto
- **Visualización de datos** con gráficos y métricas de KPIs
- **Navegación suave** entre secciones

## Secciones

1. **Hero** - Presentación principal con estadísticas clave
2. **Visión** - Objetivos y metas del proyecto
3. **Módulos** - 8 módulos del ecosistema digital
4. **KPIs** - Indicadores clave de gestión con visualizaciones
5. **Stack Tecnológico** - Arquitectura y tecnologías utilizadas
6. **Roadmap** - Hoja de ruta 2023-2030 en 5 fases
7. **Proyecto Completo** - Modal con toda la documentación estilizada

## Tecnologías Utilizadas

- **Astro 5.2.5** - Framework web moderno
- **Tailwind CSS 3.4** - Framework CSS utility-first
- **TypeScript** - Superset tipado de JavaScript
- **Google Fonts (Inter)** - Tipografía moderna

## Comandos

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Compilar para producción
npm run build

# Vista previa de producción
npm run preview
```

## Características del Diseño

- **Efectos de vidrio (Glass morphism)** en tarjetas y componentes
- **Gradientes tecnológicos** en textos y fondos
- **Animaciones suaves** al hacer scroll y hover
- **Navbar fija** con cambio de estilo al hacer scroll
- **Modal full-screen** con toda la documentación
- **Tipografía moderna** con jerarquía clara

## Alineación con Agenda 2030

El proyecto está alineado con el **ODS 11** de la ONU: Ciudades y Comunidades Sostenibles, buscando crear entornos urbanos inclusivos, seguros, resilientes y sostenibles.
